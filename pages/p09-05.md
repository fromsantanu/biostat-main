## **ğŸ§¬ SEIR Model and Extensions**

### **ğŸ“– Introduction**

The **SEIR model** extends the basic SIR model by adding an **Exposed (E) compartment**, representing individuals who are infected but **not yet infectious** (incubation period).

âœ… Particularly important for diseases like **COVID-19, measles, or influenza**, where there is a **latent period before symptoms and transmission**.

---

### **ğŸ” What is the SEIR Model?**

**SEIR compartments:**

âœ… **S:** Susceptible
âœ… **E:** Exposed (infected but not yet infectious)
âœ… **I:** Infectious
âœ… **R:** Recovered or Removed

---

### **ğŸ“ Key Equations**

The model is defined by:

$$
\frac{dS}{dt} = - \beta S I / N
$$

$$
\frac{dE}{dt} = \beta S I / N - \sigma E
$$

$$
\frac{dI}{dt} = \sigma E - \gamma I
$$

$$
\frac{dR}{dt} = \gamma I
$$

Where:

* **Î² (beta):** Transmission rate
* **Ïƒ (sigma):** Rate of progression from exposed to infectious (1/incubation period)
* **Î³ (gamma):** Recovery rate
* **N:** Total population

---

### **ğŸ’¡ Basic Reproduction Number (Râ‚€)**

$$
R_0 = \frac{\beta}{\gamma}
$$

âœ… Represents **average number of secondary cases per infectious case** in a fully susceptible population.

---

### **ğŸ”¬ Practical Example**

#### **Scenario**

In a population of **1000 individuals**:

* **Initial state:** 999 susceptible, 0 exposed, 1 infectious, 0 recovered
* **Î² = 0.3**, **Î³ = 0.1**, **Ïƒ = 0.2**

âœ… Simulate disease spread over **160 days**.

---

### **ğŸ–¥ï¸ Implementation in Python**

Using **scipy.integrate.odeint**:

```python
# Install if not installed
# pip install numpy scipy matplotlib

import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

# Total population
N = 1000
I0 = 1
E0 = 0
R0 = 0
S0 = N - I0 - E0 - R0

# Parameters
beta = 0.3
gamma = 0.1
sigma = 0.2

# Time grid (days)
t = np.linspace(0, 160, 160)

# SEIR model differential equations
def seir(y, t, N, beta, sigma, gamma):
    S, E, I, R = y
    dSdt = -beta * S * I / N
    dEdt = beta * S * I / N - sigma * E
    dIdt = sigma * E - gamma * I
    dRdt = gamma * I
    return dSdt, dEdt, dIdt, dRdt

# Initial conditions vector
y0 = S0, E0, I0, R0

# Integrate SEIR equations
ret = odeint(seir, y0, t, args=(N, beta, sigma, gamma))
S, E, I, R = ret.T

# Plot results
plt.figure(figsize=(8,5))
plt.plot(t, S, label='Susceptible')
plt.plot(t, E, label='Exposed')
plt.plot(t, I, label='Infectious')
plt.plot(t, R, label='Recovered')
plt.xlabel('Days')
plt.ylabel('Number of individuals')
plt.title('SEIR Model Simulation')
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()
```

### **ğŸ”‘ Interpretation**

âœ… **Exposed compartment (E)** shows **latent cases** before they become infectious
âœ… Disease spread and peak are **delayed compared to SIR model** due to incubation period
