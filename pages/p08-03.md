### **ğŸ” Bayesian Model for Proportions**

#### **1. Likelihood**

For observing **x successes in n trials**, the likelihood is:

$$

#### **2. Prior**

The **Beta distribution** is commonly used as a prior for Î¸ because:

âœ… It is **conjugate** to the Binomial likelihood.

#### **3. Posterior**

Using **Beta-Binomial conjugacy**:

$$
Posterior = Beta(Î± + x, Î² + n - x)

### **ğŸ’¡ Practical Example**

#### **Scenario**

A clinical trial observes **20 successes in 50 patients** (40% success rate). Prior studies suggest a success rate around **30%**, modeled as **Beta(3,7)**.

âœ… What is the **posterior estimate** and **95% credible interval**?

---

#### **Stepwise Calculation**

1. **Define prior**

* Î± = 3 (prior successes)
* Î² = 7 (prior failures)

---

2. **Calculate posterior parameters**

* Posterior Î± = 3 + 20 = **23**
* Posterior Î² = 7 + 30 = **37**

âœ… Posterior distribution: **Beta(23,37)**

---

3. **Calculate posterior mean**

$$
Mean = \frac{Î±}{Î± + Î²}
= \frac{23}{60}
= 0.383
$$

âœ… **Posterior mean = 38.3%**, slightly pulled towards prior (30%).

---

4. **Calculate 95% credible interval**

Using Beta distribution quantiles.

### **ğŸ–¥ï¸ Implementation in Python**

Using **scipy.stats module**:

```python

from scipy.stats import beta

# Prior parameters
alpha_prior = 3
beta_prior = 7

# Data

n = 50

# Posterior parameters
alpha_post = alpha_prior + x
beta_post = beta_prior + n - x

# Posterior mean
posterior_mean = alpha_post / (alpha_post + beta_post)

# 95% credible interval
ci_lower, ci_upper = beta.ppf([0.025, 0.975], alpha_post, beta_post)

print(f"Posterior mean: {posterior_mean:.2%}")
print(f"95% Credible Interval: ({ci_lower:.2%}, {ci_upper:.2%})")
```
